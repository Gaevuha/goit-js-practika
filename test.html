Шпаргалка: Методи додавання елементів у DOM

1. append()

Додає елемент у кінець всередині іншого елемента.

const div = document.querySelector('#container');
const p = document.createElement('p');
p.textContent = 'Це додано через append()';
div.append(p);

2. prepend()

Додає елемент на початок всередині іншого елемента.

const div = document.querySelector('#container');
const p = document.createElement('p');
p.textContent = 'Це додано через prepend()';
div.prepend(p);

3. before()

Додає елемент безпосередньо перед зазначеним елементом.

const target = document.querySelector('#target');
const p = document.createElement('p');
p.textContent = 'Це додано через before()';
target.before(p);

4. after()

Додає елемент безпосередньо після зазначеного елемента.

const target = document.querySelector('#target');
const p = document.createElement('p');
p.textContent = 'Це додано через after()';
target.after(p);

5. insertBefore()

Додає елемент перед reference-елементом у межах одного батьківського елемента.

const parent = document.querySelector('#list');
const reference = document.querySelector('#item2');
const li = document.createElement('li');
li.textContent = 'Новий елемент перед другим';
parent.insertBefore(li, reference);

6. innerHTML

Додає HTML у вигляді рядка. Не рекомендується для вмісту від користувача через XSS.

const div = document.querySelector('#content');
div.innerHTML += '

Додано через innerHTML

';

7. insertAdjacentHTML()

Додає HTML-рядок у певну позицію відносно елемента.

const div = document.querySelector('#box');
div.insertAdjacentHTML('beforeend', '

HTML через insertAdjacentHTML

');

8. insertAdjacentElement()

Додає готовий DOM-елемент у певну позицію відносно елемента.

const div = document.querySelector('#box');
const p = document.createElement('p');
p.textContent = 'Елемент через insertAdjacentElement';
div.insertAdjacentElement('afterbegin', p);

9. insertAdjacentText()

Додає простий текст у певну позицію відносно елемента.

const div = document.querySelector('#box');
div.insertAdjacentText('beforeend', 'Текст через insertAdjacentText');
